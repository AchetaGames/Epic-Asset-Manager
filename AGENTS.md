## OVERVIEW

Unofficial Epic Games Store desktop client for managing Unreal Engine assets, projects, plugins, and engine installations. Rust + GTK4/Libadwaita + Meson build system + Diesel/SQLite.

## CONVENTIONS

- **CompositeTemplate pattern**: Every widget has `imp` module with `#[derive(CompositeTemplate)]`, paired .ui XML file
- **Message passing**: All async results flow through `Msg` enum ‚Üí `update.rs` dispatcher ‚Üí UI updates on main thread
- **GObject properties**: Data models use `RefCell` + `ParamSpec` + getter/setter dispatch (not direct field access)
- **Thread pools**: CPU-bound work uses `threadpool` crate, results sent via `async_channel`
- **Platform guards**: `#[cfg(target_os = "linux")]` for Secret Service, gettext, ashpd, zbus, ghregistry
- **OnceCell initialization**: Parent references (window, download_manager) set once via `set_window()` pattern
- **Generated files**: `config.rs` and `schema.rs` are generated ‚Äî never edit directly

## ANTI-PATTERNS (THIS PROJECT)

- **NEVER edit `src/config.rs`** ‚Äî generated from `src/config.rs.in` by Meson
- **NEVER edit `src/schema.rs`** ‚Äî generated by Diesel migrations
- **Cargo.lock is now committed** ‚Äî added in PR #327 (was previously gitignored)
- **Infinite loop risk** in `download_manager/asset.rs:462` ‚Äî re-download without termination guard
- **Library/FAB duplication** ‚Äî `library/` and `fab/` are near-identical; changes to one usually need mirroring in the other
- **Deprecated GTK widgets** in .ui files: GtkInfoBar (‚Üí AdwBanner), GtkComboBoxText (‚Üí GtkDropDown), AdwPreferencesWindow (‚Üí AdwPreferencesDialog)
- **No test suite** ‚Äî no `#[test]`, no `tests/` directory, no `cargo test` in CI

## UNIQUE STYLES

- **Dual build system**: Meson (primary) invokes Cargo via `build-aux/cargo.sh`
- **Flatpak-first**: Development uses `fenv` for isolated Flatpak builds
- **Deep linking**: Handles `com.epicgames.launcher://` URI scheme in `application.rs::open()`
- **Bundled SQLite**: `libsqlite3-sys` with `bundled` feature ‚Äî no system SQLite dependency
- **GResource bundling**: All .ui templates compiled into binary via `resources.gresource.xml`
